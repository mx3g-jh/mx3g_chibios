Abstract Persistent Storage Interface
HAL » Interfaces and Classes » Streams
Collaboration diagram for Abstract Persistent Storage Interface:

Detailed Description

This module define an abstract interface for generic persistent storage. Such storage has a fixed size and can be read and written.
